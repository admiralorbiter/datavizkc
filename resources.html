It also has tons of online resources and open source visualizations. However, we are more than happy to have teams submit projects using other tools. Tutorials

These sites have tutorials that will help you get started with your visualization. You can find tutorials for many different tools.

Data To Viz
From Data to Viz leads you to the most appropriate graph for your data. It links to the code to build it and lists common caveats you should avoid.
DataViz Project
This is a website trying to present all relevant data visualizations, so you can find the right visualization and get inspired how to make them.
